## Clone Repository

Each topic in this document references the source code contents, which should be
cloned to a personal workspace.

```shell--macos-linux
git clone https://github.com/oracle-quickstart/oci-cloudnative.git
cd oci-cloudnative
```

```shell--win
git clone https://github.com/oracle-quickstart/oci-cloudnative.git
dir oci-cloudnative
```
